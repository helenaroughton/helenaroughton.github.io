What is display inline?

Display inline changes the positioning of elements so that they flow in the same horizontal line together rather than following after one other on the next line down. The limitations of display inline are that you can't alter the width or height of the element and you can only alter the horizontal padding, margin, or border of the element. Inline is more suited to highlighting/bordering text within a sentence. See the image below for how my navigation tiles looked with just display inline. I had attempted to add margin and padding here which had zero effect and you can see the elements hug each other like they're in the same line of text.

What is display inline-block?

Display inline-block also changes the positioning of elements so that they flow in the same horizontal line together as other elements with inline or inline-block tags. The big difference here is that the element is also treated like a block element and you can alter positioning and sizing such as padding, margin, borders, width and height. This blew my mind when I learned this key difference. The image below shows inline-block and you can see already the elements are treated more as seperate blocks.

This next image shows the elements as inline block with some padding, a height of 150px, margin on the left of 15px, the bottom 15px and width set to auto. You can see that width-auto wraps around the size of the text in the box.
